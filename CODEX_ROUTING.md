# Codex Routing para InvitaSR

Este archivo ya no es la documentacion extensa. Queda como puente corto para que cualquier agente o persona encuentre rapido la capa correcta.

## Fuente de verdad actual

1. `AGENTS.md`
2. `.codex/ROUTING.md`
3. `AI_CONTEXT.md`
4. `CLAUDE.md`

## Uso correcto

- `.codex/ROUTING.md`: regla operativa de clasificacion, ownership, skills y subagentes
- `.codex/README.md`: documentacion humana completa, ejemplos de prompts y mantenimiento
- `AGENTS.md`: principios de producto + politica minima obligatoria que siempre debe prevalecer

## Regla practica

Si hay duplicacion o conflicto, mantener solo una version detallada en `.codex/README.md` y una sola version operativa en `.codex/ROUTING.md`.
